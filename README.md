# Task 1: Calculate Factorial Using a Function

# Objective-
To create a Python function that calculates the factorial of a number.

# Program Features-
A function named factorial() is defined.
The function takes a number as an argument.
It calculates the factorial using a loop.
The calculated factorial value is returned.
The function is called with a sample number and the result is printed.

# Example Output-
If the input number is 5, the output will be:
Factorial of 5 is: 120
